HTML Basics Tutorial

HTML (HyperText Markup Language) is the standard language for creating web pages.

1. Basic Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    Content goes here
</body>
</html>

2. Headings

Use <h1> to <h6> for headings:

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>

3. Paragraphs

<p>This is a paragraph.</p>

4. Links

<a href="https://www.example.com">Visit Example</a>

5. Images

<img src="image.jpg" alt="Description">

6. Styling HTML with CSS

You can style your HTML using CSS (Cascading Style Sheets) in three main ways:

  • Inline CSS: Add the style attribute directly to an HTML element.
    <p style="color:blue;">This is blue text.</p>

  • Internal CSS: Place CSS rules inside a <style> tag in the <head> section.
    <style>
  p { color: blue; }
</style>
  • External CSS: Link to a separate CSS file using the <link> tag in the <head>.
    <link rel="stylesheet" href="styles.css">

External CSS is the most common and recommended way, as it keeps your code organized and reusable across multiple pages.

7. Components Used in This HTML Page

This page uses several important HTML components to organize content and create an interactive learning experience:

  • <!DOCTYPE html> and <html>: Declare the document type and start the HTML document.

  • <head>: Contains meta information, the page title, links to the favicon and external CSS for styling.

  • <nav> and <ul>: The navigation bar uses an unordered list of links (<a>) to help users move between pages.

  • <section class="dashboard">: Groups the main content of the page.

  • <div class="dashboard_content">: Contains all the tutorial information and interactive elements.

  • <div class="intro"> and <div class="info">: Present the introduction and step-by-step HTML basics, using headings (<h2>, <h3>), paragraphs (<p>), code examples (<pre>, <code>), and lists (<ul>, <li>).

  • <div class="playground">: An interactive area where users can type HTML code in a <textarea>, click a button to run it, and see the result live in an <iframe>.

  • <script>: JavaScript that updates the preview area with the user’s HTML code.

Together, these components create a structured, easy-to-follow tutorial page with both explanations and hands-on practice!
