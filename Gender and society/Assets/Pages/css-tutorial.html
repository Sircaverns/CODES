CSS Basics Tutorial

CSS (Cascading Style Sheets) is used to style and layout web pages.

1. How to Add CSS

There are three ways to add CSS to your HTML:

  • Inline: <h1 style="color:blue;">Hello</h1>
  • Internal: 
    <style>
  h1 { color: blue; }
</style>
  • External: 
    <link rel="stylesheet" href="styles.css">

2. Basic Selectors

h1 {
  color: blue;
}
p {
  font-size: 16px;
}

3. Example: Styling a Button

button {
  background-color: #472ca1;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 10px;
}

4. CSS Components Used in This Page

This page uses several important CSS components to create a modern, responsive, and visually appealing layout:

  • :root — Defines CSS variables (custom properties) for colors and other values, making it easy to keep your design consistent.

  • body — Sets the background color for the whole page.

  • nav, a, .blk:hover, a:hover — Styles the navigation bar and links, including hover effects for interactivity.

  • @media — Adds responsive design for small screens.

  • .dashboard, .dashboard_header, .dashboard_content — Uses Flexbox and gradients to create a card-like dashboard layout with a styled header and content area.

  • .intro, .info — Adds background color, padding, rounded corners, and box shadows to highlight important sections.

  • .button-group, button, button.right, button.left, button:hover — Centers and styles buttons, including color, padding, rounded corners, and hover effects.

  • .playground, .playground-container, .playground-editor, .playground-preview — Styles the interactive code playground, including the textarea, run button, and preview area.

Together, these CSS components make the page easy to use, visually attractive, and interactive for learning!
